# 2026-06-04 计算机方块各面作用说明制作记录

## 目标

继续完善差分机和分析机的可用说明，把数表打印机和机械打印机“每个面具体负责什么”标清楚。

上一版已经说明了放组件、接动力、放入物品和取出产物，但还没有直接回答玩家会遇到的问题：

```text
哪一面接动力？
哪一面右击放物品？
组件到底是贴在哪一面，还是放在周围？
```

## 设计思路

数表打印机和机械打印机现在是 `RotatedPillarKineticBlock`，也就是轴向动力方块。它们没有固定的“正面”，而是根据放置时的轴向决定哪一对相对面是动力口。

因此说明不能写死为“东面输入”或“北面输出”。本次采用更准确的写法：

- 轴向两端：动力输入/输出面。
- 非轴向四面：玩家右击操作面。
- 周围 4 格：组件扫描范围，不属于某一个面。

## 实现方式

### Tooltip

修改：

- `src/main/resources/assets/createaddon/lang/zh_cn.json`
- `src/main/resources/assets/createaddon/lang/en_us.json`

新增：

- 数表打印机 `各面作用 / Face roles`
- 机械打印机 `各面作用 / Face roles`

说明内容：

```text
轴向两端接动力。
非轴向四面用于右击操作。
周围 4 格扫描对应计算组件。
```

### Ponder

修改：

- `src/main/java/io/github/n013ody/createaddon/client/ponder/GuidanceScenes.java`

差分机和分析机 Ponder 中的动力步骤改为：

```text
Connect shaft power to the two axis-end faces. The other faces are for right-click use.
```

这样玩家按住 W 时能直接看到“轴向两端”和“其他面”的区别。

### 玩家说明文档

修改：

- `docs/使用说明-差分机与分析机.md`

为差分机和分析机分别新增“各面作用”段落：

- 轴向两端：接传动轴、齿轮箱或其他 Create 动力结构。
- 非轴向四面：右击放入输入物品和取出产物。
- 周围 4 格范围：识别组件。

### 验证脚本

修改：

- `.codex-tools/validate-computation-engines.mjs`

新增检查：

- 数表打印机 tooltip 必须包含各面作用说明。
- 机械打印机 tooltip 必须包含各面作用说明。

## 后续建议

如果还需要进一步“看一眼就懂”，下一步应进入模型和贴图层：

- 在轴向两端贴动力轴/铜环标记。
- 在非轴向操作面贴纸张槽或取出口标记。
- 在顶部做状态灯或计算窗口。
- 在 Ponder 中用箭头分别指向动力口、操作面和组件扫描范围。
