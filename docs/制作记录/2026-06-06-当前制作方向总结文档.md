## 本次总结的核心结论

1. 计算学不是独立于 Create 的新机械系统，而是建立在 Create 原有动线和应力网络上的机械计算扩展。
2. 以后所有计算结构都要先考虑动线、应力和真实连接，再考虑计算功能。
3. 差分机和分析机不能继续采用“附近扫描组件”的黑盒形式。
4. 应尽量复用 Create 原本已有方块，计算学只补计算专用模块。
5. 当前最推荐的下一步是先把差分机重构为线性主轴式机械数表机。
