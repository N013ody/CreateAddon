## Git 忽略规则调整

在旧工作树和新工程副本的 `.gitignore` 中都补充了：

```text
.codex-tools/jdk-21/
.codex-tools/blockbench-asar/
.codex-tools/temurin-jdk-21.zip
```

原因：

- `.codex-tools` 里的脚本可以保留给协作和验证使用。
- JDK、Blockbench 解包缓存和安装 zip 体积太大，不适合上传 GitHub。

同时移除了两个工作树的 `.git/info/exclude` 中对整个 `.codex-tools/` 的本地忽略规则。这样以后可以选择把小型验证脚本提交上去，而大型本地运行时仍由 `.gitignore` 排除。
