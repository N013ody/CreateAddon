# 差分机与分析机使用说明

## 差分机：打印基础加工曲线表

目标产物：

```text
基础加工曲线表
```

需要准备：

- 数表打印机
- 函数设定鼓
- 差分寄存器柱
- 空白数表纸
- Create 动力输入和足够应力容量

搭建方式：

```text
在数表打印机 4 格范围内放置：
- 至少 1 个函数设定鼓
- 至少 2 个差分寄存器柱
```

启动方式：

```text
1. 给数表打印机接上传动轴。
2. 确保动力网络没有应力过载。
3. 手持 空白数表纸 右击 数表打印机。
4. 等待计算完成。
5. 空手右击 数表打印机 取出 基础加工曲线表。
```

各面作用：

```text
轴向两端：
- 动力输入/输出面。
- 根据放置方向决定是哪一对面。
- 接上传动轴、齿轮箱或其他 Create 动力结构。

非轴向四面：
- 玩家操作面。
- 手持空白数表纸右击可以开始打印。
- 完成后空手右击可以取出基础加工曲线表。

周围 4 格范围：
- 不是某一个面，而是整台机器的组件扫描范围。
- 函数设定鼓和差分寄存器柱放在这个范围内都会被识别。
```

扩展规则：

- 函数设定鼓每个提供 30 计算单位。
- 差分寄存器柱每个提供 20 计算单位。
- 计算单位越高，打印越快。
- 转速越高，打印越快。
- 转速越高，需要的应力容量也越多。
- 动力不足或应力过载时，计算不会推进。

## 分析机：编译机器校准卡

目标产物：

```text
机器校准卡
```

需要准备：

- 机械打印机
- 穿孔卡读卡器
- Mill 运算磨坊
- Store 存储柱
- 基础加工曲线表
- Create 动力输入和足够应力容量

搭建方式：

```text
在机械打印机 4 格范围内放置：
- 至少 1 个穿孔卡读卡器
- 至少 1 个 Mill 运算磨坊
- 至少 1 个 Store 存储柱
```

启动方式：

```text
1. 给机械打印机接上传动轴。
2. 确保动力网络没有应力过载。
3. 手持 基础加工曲线表 右击 机械打印机。
4. 等待计算完成。
5. 空手右击 机械打印机 取出 机器校准卡。
```

各面作用：

```text
轴向两端：
- 动力输入/输出面。
- 根据放置方向决定是哪一对面。
- 接上传动轴、齿轮箱或其他 Create 动力结构。

非轴向四面：
- 玩家操作面。
- 手持基础加工曲线表右击可以开始编译。
- 完成后空手右击可以取出机器校准卡。

周围 4 格范围：
- 不是某一个面，而是整台机器的组件扫描范围。
- 穿孔卡读卡器、Mill 运算磨坊和 Store 存储柱放在这个范围内都会被识别。
```

扩展规则：

- 穿孔卡读卡器每个提供 25 计算单位。
- Mill 运算磨坊每个提供 40 计算单位。
- Store 存储柱每个提供 30 计算单位。
- 计算单位越高，编译越快。
- 转速越高，编译越快。
- 转速越高，需要的应力容量也越多。
- 动力不足或应力过载时，计算不会推进。

## 游戏内提示

- 按住 Shift 查看方块使用说明。
- 对方块按住 W 打开 Create 风格思考场景。
- 如果提示缺少组件，检查组件是否在打印机 4 格范围内。
- 如果提示没有动力，检查传动轴是否接到打印机轴向。
- 如果提示应力过载，增加动力源应力容量或降低转速。
- 如果不确定哪一面是轴向，看方块放置后的模型方向；轴向那一对相对面就是动力口。
